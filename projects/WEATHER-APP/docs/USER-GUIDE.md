# WEATHER-APP — 用户指南

## 简介

天气查询小工具，在浏览器中输入城市名称即可查看当前天气信息。

## 启动应用

1. 启动后端服务（详见 [部署文档](DEPLOY.md)）
2. 用浏览器打开 `frontend/index.html`

## 查询天气

1. 在页面顶部的搜索框中输入城市名称，例如"北京"、"上海"、"广州"
2. 点击「搜索」按钮，或按键盘回车键
3. 页面将显示天气卡片，包含以下信息：
   - 城市名称
   - 当前温度（℃）
   - 天气描述（如"晴"、"多云"、"小雨"）
   - 风向（如"东北风"、"西南风"）

## 错误提示说明

| 提示信息 | 原因 | 解决方法 |
|----------|------|----------|
| 请输入城市名称 | 搜索框为空 | 输入城市名后再搜索 |
| 未找到该城市 | 输入的城市名称无法识别 | 检查拼写，使用标准城市名（如"北京"而非"北京市"） |
| 天气服务暂时不可用 | 天气数据源服务异常 | 稍后重试 |
| 网络错误，请检查网络连接 | 浏览器无法连接后端服务 | 确认后端已启动，检查网络连接 |

## 支持的设备

- 电脑浏览器（Chrome、Firefox、Safari、Edge）
- 手机浏览器（页面已做响应式适配）

## 注意事项

- 本工具查询的是当前实时天气，不提供天气预报功能
- 温度单位为摄氏度（℃）
- 支持中文和英文城市名称
